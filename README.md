# 🚀 Fullstack Course Repository

This repository contains my progress, exercises, and projects from a **fullstack web development course**.
It covers both **frontend and backend development**, focusing on modern JavaScript technologies and best practices.

---

## 📚 About the Course

The course focuses on building modern web applications using:

- **Frontend:** HTML, CSS, JavaScript, React
- **Backend:** Node.js, Express
- **APIs:** RESTful services
- **Tools:** Git, npm, browser dev tools

The goal is to understand how fullstack applications work end-to-end—from user interface to server logic.

---

## 📂 Project Structure

The repository is organized by parts/modules of the course:

```
part1/
part2/
part3/
...
```

Each part contains:

- Exercises
- Small projects
- Notes and experiments

## 🛠️ Technologies Used

## 🎯 Learning Goals

- Understand **client-server architecture**
- Build **REST APIs with Express**
- Work with **forms and user input**
- Learn **state management in React**
- Connect frontend and backend
- Improve debugging and problem-solving skills
